## 命名规范

1.文件夹命名以及文件命名以中划线命名

> xxx-xx/xxx-xx.vue

2.页面中的 class 以下划线命名

> 采用 BEM 风格

3.函数名和变量名以驼峰命名法命名

> searchForm
